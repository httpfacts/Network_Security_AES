Network Security — AES Demo

Topic: AES (Advanced Encryption Standard)

AES is a symmetric block cipher commonly used to secure data. This mini-project implements client-side AES-GCM encryption and decryption using the browser Web Crypto API.

What this demo does

  • Derives an AES-256 key from a password using PBKDF2 (with random salt).
  • Encrypts text with AES-GCM using a random IV.
  • Packs salt + IV + ciphertext into a single Base64 string for easy sharing/storage.
