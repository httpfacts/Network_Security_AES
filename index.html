Topic: AES (Advanced Encryption Standard)

AES is a symmetric key encryption algorithm widely used for secure data transmission. It operates on fixed block sizes (128-bit block) using key sizes of 128, 192, or 256 bits.

Key points

  • Type: Symmetric block cipher
  • Block size: 128 bits
  • Key sizes: 128 / 192 / 256 bits
  • Mode used in demo: AES-GCM (authenticated encryption)

Why AES-GCM?

AES-GCM (Galois/Counter Mode) provides confidentiality and integrity (authentication). It's recommended over older modes (like ECB) because it avoids pattern leakage and provides a message authentication tag.
